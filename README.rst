Montafon
========

The website of Eloquentlog_.


Build
-----

.. code:: zsh

    % cargo build --release


Development
~~~~~~~~~~~

Use cargo-watch_

.. code:: zsh

    % cargo watch -x 'run' -d 0.3


Testing
~~~~~~~

.. code:: zsh

    % cargo test
